Dataset Overview & Metadata
Introduction and dataset dimensions

This interactive lab provides a detailed data-science walkthrough on university netflow telemetry records. Telemetry logs are aggregated at 10-minute, 1-hour, and 1-day increments to optimize computation scales.

We use machine learning to segment subnets by behavioral profiles (e.g., distinguishing web servers from dorm computers), forecast future capacities, and detect cyber-attacks like scanning or data exfiltration.
